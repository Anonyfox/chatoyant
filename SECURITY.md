# Security Policy

## Supported versions

Security fixes are applied to the latest released version of `chatoyant`. Please
upgrade to the most recent release before reporting an issue.

## Reporting a vulnerability

Please report security vulnerabilities privately — do not open a public issue.

- Preferred: open a [GitHub security advisory](https://github.com/Anonyfox/chatoyant/security/advisories/new).
- Alternatively, email **max@anonyfox.com** with details and reproduction steps.

You can expect an acknowledgement within a few business days. Once a fix is
released, we will credit reporters who wish to be named.
